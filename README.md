# Walmart-Sales-Analysis
Analyzed Walmart sales data to uncover sales trends, customer purchasing patterns, branch performance, and revenue insights.

## Objective
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Answer key business questions using SQL
- Create interactive visualizations and dashboards

## Tools & Technologies
- Python
- Pandas
- NumPy
- SQL
- Tableau
- Jupyter Notebook

## Key Business Questions
1. Which branch generated the highest revenue?
2. What product categories performed best?
3. Which customer type contributes most to sales?
4. What are the peak sales periods?
5. How do customer ratings vary across branches?

## Key Insights
- Identified top-performing branches and product lines.
- Analyzed customer purchasing behavior.
- Found peak sales days and time periods.
- Generated actionable business recommendations.
